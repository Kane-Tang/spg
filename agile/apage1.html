Zoho Sprints Tutorial

  1. Go to "https://www.zoho.com/sprints/" and create account.
  2. Login to your account.
  3. Select "+project" in home page.








  4. In "Plan" taskbar -> select "Create WorkItems" to create modules eg. Planning, Design, Code, Testing.




  5. Next select "Create Sprint" and add the WorkItems into the Sprint.




  6. Select "Start" near the created Sprint.




  7. Go to "Board" -> Drag and Drop the Modules into respective status. Modules should be moved as per the current status of the software.



  8. In "Reports" -> Select appropritate report to see change in schedule and backlogs.

    Required Reports :

    • Cumulative Flow Chart



    • BurnDown Chart using "Count" option under ViewBy.
      Note: No need to use compare option as you will be implementing only one sprint.



  9. These reports must be included along with the final report to be accounted for grading.

  10. Reports are automatically created when you move the work items you created in the "Board" tab. So you have update the work items promptly in the Zoho Board to get accurate reports.
