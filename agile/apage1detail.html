Deliverables for each Sprint

  1. User Story: A detailed user story and todo items.
  2. Need visual proof of the completetion of the Sprint, including coding, unit testing and code integration. To do this go to the BOARD tab in Zoho SPRINT and show the entire history.

  3. Reports of the implementation is recorded automatically. To get this go to the REPORT tab in Zoho SPRINT and show the entire project report for the Sprint including cumulative flowchart, Burndown chart and so on.
